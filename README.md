# Small-Office-Network-PacketTracer
Cisco Packet Tracer simulation for a multi-department office network using RIP routing.

This project involves the design and configuration of a hierarchical network for a small office consisting of Sales and HR departments using Cisco Packet Tracer.

## Key Features:
* **Topology:** 3-router architecture (Sales, HR, and Central Backbone).
* **Routing:** Configured using RIP (Routing Information Protocol).
* **Connectivity:** Tested and verified via Ping commands between all departments.
